# Observo

Observo is a Django web application developed as part of a university project (Framework Web 1)

The application allows users to manage animals and record observations of animals in the wild. Each observation includes information such as the observed species, date and time, GPS location, and a textual description.

## Features
- Animal database with taxonomic classification
- Record animal observations
- View observation details
- Manage data through Django admin interface

## Technologies
- Python
- Django
- SQLite
- HTML / CSS
